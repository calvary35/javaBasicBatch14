Constructors:
    Constructor help us give initial values to the fields of a class.
Why we should use a constructor:
    Constructors help us writer cleaner code.
    Constructors give us more control over the values that an end user can assign to the field of a class.
Syntax of Constructor:
    Access modifier className (parameters){

    }

   String name;

   name=dogName;
